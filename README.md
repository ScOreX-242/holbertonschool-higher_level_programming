# holbertonschool-higher_level_programming

## Description

This repository contains Python projects for the Holberton School Higher Level Programming curriculum. Each directory represents a separate project covering different aspects of Python programming.

## Projects

| Project | Description |
| --- | --- |
| [python-hello_world](./python-hello_world) | Introduction to Python: print, strings, and basic syntax |
| [python-if_else_loops_functions](./python-if_else_loops_functions) | Control flow: if/else, loops, and functions |
| [python-import_modules](./python-import_modules) | Modules, imports, and command-line arguments |
| [python-data_structures](./python-data_structures) | Lists, tuples, and data structure manipulation |
| [python-more_data_structures](./python-more_data_structures) | Sets, dictionaries, lambda, map, filter, reduce |
| [python-exceptions](./python-exceptions) | Exception handling with try/except/finally |
| [python-classes](./python-classes) | Object-Oriented Programming: classes and objects |
| [python-test_driven_development](./python-test_driven_development) | Test-driven development with doctests and unittests |
| [python-more_classes](./python-more_classes) | Advanced OOP: class methods, static methods, and special methods |
| [python-everything_is_object](./python-everything_is_object) | Python object model, identity, and mutability |
| [python-inheritance](./python-inheritance) | Class inheritance and polymorphism |
| [python-input_output](./python-input_output) | File I/O, JSON serialization and deserialization |
| [python-almost_a_circle](./python-almost_a_circle) | Comprehensive OOP project: Base, Rectangle, and Square classes |

## Requirements

- Python 3.8+
- Ubuntu 20.04 LTS
- pycodestyle 2.7.*

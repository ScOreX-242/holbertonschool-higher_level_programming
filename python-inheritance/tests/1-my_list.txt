The ``MyList`` class
====================

Using ``MyList``
----------------

Importing the class from the module:

    >>> MyList = __import__('my_list').MyList

Creating an empty MyList:

    >>> my_list = MyList()
    >>> print(type(my_list))
    <class 'my_list.MyList'>

Printing an empty list sorted:

    >>> my_list.print_sorted()
    []

Creating a MyList with one element:

    >>> my_list = MyList([5])
    >>> my_list.print_sorted()
    [5]

Creating a MyList with multiple elements:

    >>> my_list = MyList([3, 1, 2])
    >>> my_list.print_sorted()
    [1, 2, 3]

Original list should remain unchanged:

    >>> my_list = MyList([3, 1, 2])
    >>> print(my_list)
    [3, 1, 2]
    >>> my_list.print_sorted()
    [1, 2, 3]
    >>> print(my_list)
    [3, 1, 2]

Already sorted list:

    >>> my_list = MyList([1, 2, 3, 4])
    >>> my_list.print_sorted()
    [1, 2, 3, 4]

Reverse sorted list:

    >>> my_list = MyList([5, 4, 3, 2, 1])
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

List with duplicate elements:

    >>> my_list = MyList([2, 3, 2, 1, 1])
    >>> my_list.print_sorted()
    [1, 1, 2, 2, 3]

List with negative integers:

    >>> my_list = MyList([-3, -1, -2, 0])
    >>> my_list.print_sorted()
    [-3, -2, -1, 0]

Mixed positive and negative integers:

    >>> my_list = MyList([10, -1, 5, 3])
    >>> my_list.print_sorted()
    [-1, 3, 5, 10]
